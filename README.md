# SafeMode9
This is a compiled SafeMode9 repository for 3ds.

i will try to update this as much as i can, it might come out a little bit later than the official updates of GodMode9 considering i have to compile all future updates.

if you didnt know SafeMode9 is a varient of GodMode9 that is completely safe to use with no brick risks at all, i recommend this for beginners as GodMode9 might be a bit risky for new 3ds modders.
